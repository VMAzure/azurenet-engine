# azurenet-engine

Dedicated data synchronization engine for vehicle data.

## Responsibilities
- Sync auto nuove
- Sync auto usate
- Sync VIC nuovo
- Sync VIC usato

## Explicitly NOT included
- No API runtime
- No frontend
- No AI
- No images or videos
- No notifications
- No billing
- No SEO

## Architecture
- Shared PostgreSQL (same DB as CoreAPI)
- Process isolation
- Scheduler-based batch jobs only

This repository is part of the AZCORE ecosystem.
